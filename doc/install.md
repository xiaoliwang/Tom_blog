Getting started
----

###Pre-requisites

- [Git][1]
- [iojs/npm][2]
- [Bower][3]

### Before debugging

- Download development toos:
> npm install

- Download dependencies: 
> bower install

- rename config.example.js
> rename config.example.js config.js

- Start at http://localhost:3000/
> npm start
